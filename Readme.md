# NLP Final Project - 宏哥話生成器

## 概念

透過句子上有反義詞概念的名詞進行捕捉與預測，重新生成句子

## 製作方法

已寫在簡報內  

## 使用方法

```
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```
並確保已安裝 `CUDA` 讓 `GPU` 可以使用  
以及有 `Google API Key` ， 可以執行 `Gemini`    
之後按照順序執行 `antithesis-generation.ipynb` 即可
